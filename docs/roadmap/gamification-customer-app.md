## Overview

This document specifies the customer-facing mobile app experience for the gamification system, building upon the original pirate theme spec.

---

## Navigation Structure

| Tab | Purpose |
|-----|--------|
| Home | Dashboard, activity, quick actions |
| My Code | QR code for check-in |
| Explore | Map, locations, voyages, events |
| Loot | Rewards, collection, redemption |
| Profile | Stats, crew, settings |

---

## Key Screens

### Home (Captain's Log)
- Rank and XP progress
- Tenant point balances
- Active bounties
- Upcoming events
- Recent activity

### Explore
- Map with nearby locations
- Active raids highlighted
- Event markers
- List/filter view

### Collection
- Grid of captured creatures
- Rarity sections
- Completion progress
- Missing creature hints

### Crew
- Crew stats and rank
- Leaderboard position
- Active challenges
- Member list
- Chat

### Profile
- Avatar and rank display
- Lifetime stats
- Badge showcase
- Settings

---

## Animation Guidelines

### Haptic Feedback

| Event | Haptic |
|-------|--------|
| Check-in success | Medium impact |
| Capture success | Success notification |
| Rank up | Heavy impact |

### Sound Effects

| Event | Sound |
|-------|-------|
| Check-in | Coin clink |
| Creature spawn | Alert chime |
| Capture | Treasure chest |
| Rank up | Fanfare |

---

## Accessibility

- VoiceOver/TalkBack support
- 4.5:1 minimum contrast
- 44x44pt minimum touch targets
- Reduce motion support
- Don't rely on color alone

## Database Schema

```sql
-- Platform-wide feedback
CREATE TABLE platform_feedback (
  id TEXT PRIMARY KEY,
  customer_id TEXT NOT NULL,
  feedback_type TEXT NOT NULL,        -- 'review', 'suggestion', 'problem', 'question', 'compliment'
  rating INT,                          -- 1-5, NULL if not a review
  feedback_text TEXT NOT NULL,
  loved_features TEXT[],               -- What they love (reviews)
  category TEXT,                       -- 'app_features', 'rewards', 'businesses', etc.
  testimonial_consent BOOLEAN DEFAULT FALSE,
  contact_consent BOOLEAN DEFAULT FALSE,
  device_info JSONB,                   -- OS, app version, device
  status TEXT DEFAULT 'new',           -- 'new', 'read', 'replied', 'archived', 'flagged'
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Admin responses to feedback
CREATE TABLE feedback_responses (
  id TEXT PRIMARY KEY,
  feedback_id TEXT NOT NULL REFERENCES platform_feedback(id),
  admin_user_id TEXT NOT NULL,
  response_text TEXT NOT NULL,
  sent_via TEXT[],                     -- ['in_app', 'email']
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Published testimonials
CREATE TABLE testimonials (
  id TEXT PRIMARY KEY,
  feedback_id TEXT NOT NULL REFERENCES platform_feedback(id),
  display_text TEXT NOT NULL,          -- May be edited from original
  display_name TEXT NOT NULL,          -- "Sarah M."
  display_location TEXT,               -- "Des Moines, IA"
  rating INT NOT NULL,
  placement TEXT[],                    -- ['hero', 'customer_section', 'social']
  hero_order INT,                      -- Order in hero section (1, 2, 3)
  status TEXT DEFAULT 'draft',         -- 'draft', 'published', 'archived'
  published_at TIMESTAMPTZ,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Feedback incentives tracking
CREATE TABLE feedback_rewards (
  id TEXT PRIMARY KEY,
  customer_id TEXT NOT NULL,
  feedback_id TEXT NOT NULL,
  points_awarded INT NOT NULL,
  reason TEXT,                         -- 'review_submitted', 'testimonial_used'
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE INDEX idx_feedback_type ON platform_feedback(feedback_type, status);
CREATE INDEX idx_feedback_rating ON platform_feedback(rating) WHERE rating IS NOT NULL;
CREATE INDEX idx_testimonials_status ON testimonials(status);
```

## Tenant Review Distribution

### The Opportunity

When customers rate a business 4-5 stars with a comment, tenants can push those reviews to external platforms:

- **Google My Business** - SEO boost, local search
- **Trustpilot** - Trust signals
- **Yelp** - Discovery
- **Facebook** - Social proof
- **Website widget** - On-site testimonials
- **WooCommerce/Shopify** - Product reviews

**This is a major tenant incentive** - positive reviews from the rewards program become external marketing fuel.

{
  platform: "google_my_business",
  credentials: { ... }  // OAuth tokens
}

// Request customer to post on external platform
POST /api/admin/reviews/:id/request-external
{
  platforms: ["google", "facebook"],
  customMessage: "Thanks for your review!"  // Optional
}

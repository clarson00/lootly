## What is a Voyage?

A **Voyage** (quest/challenge) is a multi-step journey with a big reward at the end.

| Concept | Pirate Term | Example |
|---------|-------------|---------|
| Quest | Voyage | "Weekend Warrior" |
| Steps | Voyage Legs | "Visit 3 weekends in a row" |
| Progress | Treasure Map | 2/3 legs complete |
| Completion | Conquer | Voyage conquered! |
| Reward | Grand Treasure | 200 bonus doubloons |

---

## Voyage Types

### Time-Based Voyages

| Type | Example | Structure |
|------|---------|-----------|
| **Streak** | Visit 5 days in a row | Sequential, no breaks |
| **Period** | Visit 3 times this month | Any order, deadline |
| **Countdown** | Complete before Sunday | Race against time |

### Action-Based Voyages

| Type | Example | Structure |
|------|---------|-----------|
| **Multi-location** | Visit 3 different locations | Any order |
| **Spend threshold** | Spend $100 total | Cumulative |
| **Item-based** | Order from 3 categories | Specific actions |
| **Combo** | Visit + spend + invite friend | Mixed requirements |

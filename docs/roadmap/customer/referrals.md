## Overview

Allow customers to refer friends and earn rewards. Phone-based flow that matches our brick-and-mortar focus.

## Key Concepts

### Phone-First Referrals
- Customer gets unique referral code or link
- "Text this to a friend" mechanic
- Friend signs up via SMS or at checkout
- Both parties earn rewards when friend makes first purchase

### Reward Types
- Points for referrer
- Points for referee (welcome bonus)
- Unlockable rewards
- Multiplier boosts

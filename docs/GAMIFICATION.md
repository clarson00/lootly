## Visual Design Guidelines

### Color Palette

| Element | Color | Hex |
|---------|-------|-----|
| Primary (Gold) | Doubloons, highlights | #F59E0B |
| Secondary (Teal) | Ocean, secondary actions | #14B8A6 |
| Dark (Navy) | Backgrounds | #0F172A |
| Accent (Red) | Alerts, urgency | #EF4444 |
| Success (Green) | Completed, earned | #22C55E |

### Typography

- **Headers**: Bold, slightly pirate-y (but still readable)
- **Body**: Clean, modern sans-serif
- **Numbers**: Prominent, easy to read at a glance

### Animations

- Coin animations for earning doubloons
- Map path animations for voyage progress
- Treasure chest opening for rewards
- Confetti burst for achievements
- Ship sailing animation for journey progress
